Chapter 17

AI Integration

Ollama Setup

Ollama provides local AI capabilities. Install and configure:

  1. Install Ollama from ollama.ai
  2. Pull a model: ollama pull qwen3:1.7b
  3. Start Ollama: ollama serve
  4. Configure in Ronin: export OLLAMA_URL="http://localhost:11434"

Model Selection

Choose models based on your needs:

  • qwen3:1.7b - Fast, lightweight (default)
  • llama2 - Good balance
  • mistral - High quality

Remote AI Providers

Grok (xAI)

Set GROK_API_KEY environment variable or use ronin config --grok-api-key.

Gemini (Google)

Set GEMINI_API_KEY environment variable or use ronin config --gemini-api-key.

Basic AI Operations

complete()

Text Completion
const response = await this.api.ai.complete("Hello!");
console.log(response);

stream()

Streaming
for await (const chunk of this.api.ai.stream("Tell me a story")) {
  process.stdout.write(chunk);
}

chat()

Chat with History
const response = await this.api.ai.chat([
  { role: "user", content: "Hello!" },
  { role: "assistant", content: "Hi there!" },
  { role: "user", content: "How are you?" }
]);
