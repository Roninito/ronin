Troubleshooting & Debugging

Common Issues

Agent Not Running

  • Check that static schedule is defined correctly
  • Verify cron expression has exactly 5 fields
  • Ensure Ronin server is running
  • Check console logs for registration messages

Ollama Connection Error

  • Verify Ollama is running: curl http://localhost:11434/api/tags
  • Check OLLAMA_URL environment variable
  • Start Ollama: ollama serve

Plugin Not Found

  • Check file is in plugins/ directory
  • Verify default export exists
  • Check console for error messages

Memory/Database Issues

  • Check database file permissions
  • Verify database path is correct
  • Check disk space

Debugging Agents

Manual Execution

ronin run my-agent

Logging

Debug Logging
async execute(): Promise<void> {
  console.log("Agent starting...");
  try {
    const result = await this.api.ai.complete("Hello");
    console.log("Result:", result);
  } catch (error) {
    console.error("Error:", error);
  }
}

Network Issues

  • Check firewall settings
  • Verify ports are accessible
  • Test network connectivity
  • Check DNS resolution

Getting Help

  • Check documentation: ronin docs
  • Ask questions: ronin ask "your question"
  • Review logs for error messages
  • Check GitHub issues
