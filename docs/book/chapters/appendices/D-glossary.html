Appendix D

Glossary

Key Terms and Concepts

Agent
A TypeScript/JavaScript class that extends BaseAgent and implements an execute() method. Agents are the executable units of work in Ronin.
BaseAgent
The base class that all agents extend. Provides access to the API and lifecycle management.
Cron Expression
A time-based schedule expression using the format: minute hour day month weekday.
Call Sign
A unique identifier for a Ronin instance in the Realm network.
Function Calling
The ability for AI agents to use plugins as tools, making intelligent decisions about which tools to use.
Memory
SQLite-based persistent storage for agent state and conversation history.
Plugin
A reusable module that extends Ronin's capabilities. Plugins are automatically discovered and made available to agents.
Realm
A peer-to-peer communication system that enables Ronin instances to communicate with each other.
Realm Discovery Server
A central registry for peer discovery in the Realm network.
Tool
A plugin method that has been converted to a tool definition for AI function calling.
Webhook
An HTTP endpoint that triggers an agent when a POST request is received.
